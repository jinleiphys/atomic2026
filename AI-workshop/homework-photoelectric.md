## 第一题：让 AI 从密立根数据中重新发现光电方程（35 分）

1916年，密立根发表了他对光电效应的精密测量结果（[R.A. Millikan, *Phys. Rev.* 7, 355 (1916)](https://doi.org/10.1103/PhysRev.7.355)）。这个实验的设计极其巧妙：在超高真空中，用刀片刮去碱金属表面的氧化层以获得清洁表面，然后用已知频率的紫外光照射，测量遏止电压。密立根花了十年时间完善这个实验——讽刺的是，他最初是想**否定**爱因斯坦的光量子假设，结果反而提供了最有力的证据，也因此获得了1923年诺贝尔物理学奖。

密立根的数据精确地验证了 $eV_s = h\nu - \phi$ 这一关系，其中 $V_s$ 是遏止电压，$\nu$ 是光频率，$\phi$ 是金属的功函数。对于不同金属，$V_s$-$\nu$ 曲线都应接近直线，且斜率在实验误差范围内一致——这个共同斜率对应 $h/e$，即普朗克常数与电子电荷之比。斜率的普适性意味着 $h$ 是自然界的基本常数，而非任何特定材料的性质。

本题要求你先用传统方法分析数据，然后**假装不知道任何公式**，让符号回归算法从实验数据中自动发现这个关系。

### 本题你真正要回答的问题

1. 为什么不同金属的 $V_s$-$\nu$ 图像都近似是直线？
2. 为什么“不同材料共享同一斜率”这件事很重要？
3. AI 可以重新找出这个公式，但它能不能替代爱因斯坦的物理解释？

### 1.2 符号回归：让 AI "重新发现" 光电方程（12 分）

现在假装你**不知道**光电方程。给AI一堆 $(\nu, V_s)$ 数据，看它能否自动找到 $V_s$ 与 $\nu$ 的数学关系。

**什么是符号回归？** 普通的回归（比如线性回归）假定了函数形式（$y = ax + b$），然后拟合参数。符号回归不假定任何形式——它在所有可能的数学公式中搜索，找到既简洁又精确的表达式。PySR 是一个基于遗传算法的符号回归库（`pip install pysr`）。

**要求：**

(a) 选取铯（Cs）的数据，运行 PySR：

```python
from pysr import PySRRegressor

model = PySRRegressor(
    niterations=200,
    binary_operators=["+", "-", "*", "/"],
    unary_operators=["exp", "log", "square", "sqrt", "abs"],
    maxsize=15,           # 公式最大复杂度
    populations=30,       # 搜索种群数
)

nu_cs = photoelectric_data['Cs']['nu'].reshape(-1, 1)
Vs_cs = photoelectric_data['Cs']['V_s']
model.fit(nu_cs, Vs_cs)
print(model)  # 打印 Pareto 前沿：不同复杂度下的最佳公式
```

PySR 会输出一张表，列出从最简单到最复杂的多个候选公式。报告这张表，并绘制"复杂度 vs 误差"图（横轴为公式复杂度，纵轴为均方误差）。最简单的"好"公式是什么？它和线性关系一致吗？

这一步的重点不是“找到最花哨的公式”，而是观察：**当公式复杂度增加时，误差是如何下降的？最早出现的、已经足够好的公式是什么？**

### 1.3 多金属联合回归：发现"普适斜率"（8 分）

前面你对每种金属分别做了回归，然后手动比较斜率。能不能让 AI **自己发现**"所有金属斜率相同"？

**要求：**

(a) 把所有金属的数据合并。**注意：金属种类是类别变量，不应简单编码成 1 到 5 的连续数字**，否则算法可能误以为“编号越大，物理性质越大”。因此这里使用 one-hot 编码：频率 $\nu$ 作为一个连续输入，再用5个0/1变量表示金属种类。

```python
# 合并所有金属的数据
metal_names = list(photoelectric_data.keys())
nu_all, Vs_all, onehot_all = [], [], []

for name, data in photoelectric_data.items():
    for nu_i, Vs_i in zip(data['nu'], data['V_s']):
        nu_all.append(nu_i)
        Vs_all.append(Vs_i)
        onehot_all.append([1 if name == m else 0 for m in metal_names])

X_all = np.column_stack([nu_all, np.array(onehot_all)])
y_all = np.array(Vs_all)

# 运行 PySR，输入变量：x0 = ν, x1...x5 = 各金属的 one-hot 标记
model_all = PySRRegressor(
    niterations=300,
    binary_operators=["+", "-", "*", "/"],
    unary_operators=["square", "sqrt", "abs"],
    maxsize=20,
    populations=40,
)
model_all.fit(X_all, y_all)
print(model_all)
```

观察 PySR 的输出公式：$\nu$（即 $x_0$）前面的系数是否**不依赖**金属种类？各个 one-hot 变量（$x_1 \sim x_5$）是否主要只出现在加/减的常数项中？

## 第二题：阿秒光电效应——光电子到底要飞多久？（30 分）

### 背景

课堂上我们讲过，光电效应的一个关键实验事实是：响应时间**极短**，小于 $10^{-9}$ s。这与经典理论的预言矛盾——经典理论认为电子需要慢慢从电磁波中积累能量，响应时间应该很长。爱因斯坦用光量子假设完美解释了这一点：一个光子一次性把全部能量给一个电子，所以过程几乎是瞬间的。

但"几乎瞬间"到底是多快？$10^{-9}$ s 只是当时实验仪器的分辨率上限，真正的光电发射时间远比这短。

[2023年诺贝尔物理学奖](https://www.nobelprize.org/prizes/physics/2023/summary/)授予了 Pierre Agostini、Ferenc Krausz 和 Anne L'Huillier，表彰他们在产生**阿秒光脉冲**方面的开创性工作。一阿秒（as）= $10^{-18}$ 秒。为了感受这个尺度：**一阿秒之于一秒，就如同一秒之于宇宙的年龄（138亿年）。**

有了阿秒脉冲这把"超快时钟"，物理学家终于可以回答光电效应中最后一个未解之谜：**从光子被吸收到电子飞出，这个过程到底经历了什么、花了多长时间？**

2024年8月，Driver 等人在 *Nature* 上报道了一个惊人的结果（[T. Driver et al., *Nature* 632, 762–767 (2024)](https://doi.org/10.1038/s41586-024-07771-9)）：他们测量了**分子中 X 射线光电电离的绝对时间延迟**，发现在某些条件下，从光子被吸收到电子最终逃出分子，整个过程可以持续长达 **700 阿秒**。

### 本题你真正要回答的问题

1. 经典理论为什么会预言一个很长的等待时间？
2. 爱因斯坦说“几乎瞬时”时，到底抓住了什么本质？
3. 现代阿秒实验测到非零延迟后，是在推翻爱因斯坦，还是在细化我们对过程的理解？

### 2.1 经典理论预言的能量积累时间（10 分）

在爱因斯坦之前，物理学家用经典电磁理论理解光电效应：光是连续的电磁波，电子需要逐渐吸收波的能量，积累到足够多才能逃逸。我们先来算一下，这个经典图像预言的时间有多长。

(b) 如果用强度为 $I = 1$ W/m² 的光照射这个原子，它每秒钟拦截的功率为 $P = I \cdot \sigma$。要积累 $E = 5$ eV 的能量（一个典型紫外光子的能量），经典理论预言需要等待多长时间 $\tau = E / P$？把结果换算成秒。

(c) 编写代码，计算不同光强下的经典等待时间，绘制 $\tau$ vs $I$ 曲线（双对数坐标）：

```python
import matplotlib.pyplot as plt

a0 = 0.529e-10   # 玻尔半径 (m)
sigma = np.pi * a0**2  # 原子截面

I = np.logspace(-3, 20, 500)  # 光强从 10^-3 到 10^20 W/m²
tau = E_photon / (I * sigma)  # 经典等待时间

plt.figure(figsize=(10, 6))
plt.loglog(I, tau)
plt.xlabel('光强 I (W/m²)')
plt.ylabel('经典等待时间 τ (s)')

# 标注几个参考光强
# 请标出：月光(~0.001)、太阳光(~1400)、强激光(~10^15)、XFEL(~10^20)
# 也标出两条水平参考线：1 ns = 10^-9 s（旧实验分辨率）和 1 as = 10^-18 s

plt.show()
```

(d) 从图中读出：
- 在太阳光照射下，经典理论预言电子要等多久？

- 经典理论的问题一目了然：即使是微弱的光，实验上光电子也是**立刻**出现的。这就是爱因斯坦必须引入光量子的原因。

**结果自检：**
- 当 $I = 1$ W/m² 时，等待时间应在 **$10^2$ s** 量级。
- 在太阳光强附近（约 $1400$ W/m²），等待时间仍然远大于 $10^{-9}$ s，大约是 **$10^{-2}$ s** 量级。
- 要把经典等待时间压到 $10^{-9}$ s 以下，需要极端高的光强，大约 **$10^{11}$ W/m²** 量级。

## 第三题：机器学习预测材料功函数——从爱因斯坦到太阳能电池（35 分）

### 背景

光电效应方程 $K_{\max} = h\nu - \phi$ 中的功函数 $\phi$ 是什么？直觉上说，它就是"把一个电子从材料表面拽出来所需的最小能量"。$\phi$ 越小，越容易发生光电效应——所以密立根实验选择碱金属（铯、钠、钾），因为它们的功函数最小，可见光甚至就能打出电子。

但功函数的重要性远不止于此。它会显著影响：
- **太阳能电池与光伏器件**：电极与活性层之间的能级对齐、载流子抽取效率
- **LED和光电探测器**：电极注入/抽取载流子的难易程度，以及器件的界面设计
- **电子发射器件**：场发射显示器、电子显微镜阴极等器件的发射门槛

2024年，Schindler 等人在 *Advanced Functional Materials* 上报告（[Schindler et al., *Adv. Funct. Mater.* 2024, 2401764](https://doi.org/10.1002/adfm.202401764)），他们用机器学习对超过 50,000 种表面的功函数进行了预测，精度达到 0.09 eV——比逐个做高精度计算快得多。同年，德国 KIT 团队在 *Science* 上报道（[Wu, Torresi et al., *Science* 2024](https://doi.org/10.1126/science.ads0901)），他们用 AI 引导的逆向设计与高通量实验，仅用约 **150 次**实验就找到了性能更优的钙钛矿太阳能电池空穴传输材料，器件效率达到 **26.2%**。
