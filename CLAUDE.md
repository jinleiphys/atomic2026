## Project Overview

University-level atomic physics lecture slides (原子物理学 2026). Each chapter is a subdirectory; the root `index.html` is the landing page linking to all chapters.

## Chapter Structure

- **第一章/** — Static slides: exported PNG images + `slideshow.html` (generated by `generate_slideshow.py`)
- **第二章/** — Slidev-based interactive slides (`slides.md`), the most actively developed chapter
- **第三章/, 第四章/** — Slidev-based, early stage (same package.json pattern as 第二章)
- **AI-workshop/** — Standalone HTML demos (Millikan experiment, Balmer regression, symbolic regression)

## Build & Dev Commands

All Slidev chapters follow the same pattern. From inside a chapter directory (e.g. `cd 第二章`):

```bash
npm install              # first time setup
npx slidev              # dev server (hot reload)
npx slidev build --base /ch2/dist/   # production build for tongji server
npx slidev build --base /atomic2026/第二章/dist/  # production build for GitHub Pages
```

The `--base` path **must match** the URL path where the build will be served, otherwise SPA routing returns 404.

## Deployment

Two deployment targets:

1. **GitHub Pages** — automatic via `.github/workflows/deploy.yml` on push to `main`. Site: `https://jinlei.fewbody.com/atomic2026/`
2. **Internal server (tongji)** — manual via `./deploy-tongji.sh [--build]`. Serves on `http://192.168.50.11:8888/` using a Python SPA-aware HTTP server. Chinese directory names break with scp, so 第二章 is deployed as `ch2/dist/` on the server.

## Slidev Conventions

- **Font**: Kaiti SC (楷体) with fallback chain, set globally in slides.md frontmatter styles
- **Accent color**: `#C71585` (magenta) via `.kaiti-accent` CSS class
- **Math**: KaTeX rendering (`math: katex` in frontmatter)
- **Interactivity**: `v-click` directives for step-by-step reveals and teaching interactions
- **Layout**: Two-column layouts use `::left::` / `::right::` separators
- **Disabled slides**: Stored in `_disabled_slides.md` alongside `slides.md`

## Keynote Conversion

`key2slidev.py` extracts text, LaTeX, images, and video from `.key` files into Slidev markdown. Usage:
```bash
python key2slidev.py "原子物理第二章.key" -o ./第二章 -t "原子的量子态：玻尔模型"
```

## Pedagogical Approach

Slides emphasize scientific thinking and historical context ("回到120年前" thought experiments) over formula memorization. Content should guide students through the reasoning process, not just present results.
